# Minions Oss-projects

Project definitions, repos, version history, and health status

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-oss-projects/sdk`) — Core library for Node.js
- **Python SDK** (`minions-oss-projects`) — Python bindings
- **CLI Tool** (`@minions-oss-projects/cli`) — Command-line interface
- **Web App** — Interactive playground at [oss-projects.minions.wtf](https://oss-projects.minions.wtf)
