---
title: Installation
description: Install Minions Oss-projects packages
---

## TypeScript / Node.js

```bash
npm install @minions-oss-projects/sdk minions-sdk
```

## Python

```bash
pip install minions-oss-projects
```

## CLI

```bash
npm install -g @minions-oss-projects/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
